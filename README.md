# 🛒 E-commerce Sales Analysis

## 📌 Project Overview
This project analyzes e-commerce sales data to uncover business insights such as monthly sales trends, top-performing products, and category-wise performance.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Key Analysis Performed
- Data Cleaning & Preprocessing
- Feature Engineering (Total Sales, Month)
- Monthly Sales Analysis
- Category-wise Sales Distribution
- Correlation Analysis (Heatmap)
- Data Visualization (Bar Chart, Pie Chart, Heatmap)

---

## 📈 Key Insights
- Identified best-performing months based on revenue
- Found top-selling products
- Analyzed sales distribution across categories
- Discovered relationships between variables using correlation
